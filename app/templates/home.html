Why Our Admission System?

Designed to ensure fairness, transparency, and efficiency in the admission process

Rule-Based Automated Screening

Our intelligent inference engine evaluates candidates against predefined admission rules including UTME scores, O'Level results, subject combinations, and quota requirements - eliminating human error and bias.

  • Instant candidate evaluation
  • Consistent rule application
  • Complete reasoning trace

JAMB-Compliant Quota System

Full implementation of the federal quota system with Merit (45%), Catchment (35%), and Educationally Less Developed States - ELDS (20%) categories, ensuring equitable access to higher education.

  • Merit-based ranking
  • Automatic catchment classification
  • ELDS consideration built-in

Comprehensive Analytics Dashboard

Monitor admission progress with real-time statistics, quota distribution charts, rejection reason analysis, and state-of-origin demographics - all exportable for Senate reporting.

  • Interactive charts and graphs
  • Export to Excel and PDF
  • Audit trail for every decision

JAMB Format CSV Import

Seamlessly import candidate data directly from JAMB CAPS in the standard CSV format. Automatic format detection, validation, and mapping to system database with error reporting.

  • Drag-and-drop file upload
  • Automatic format recognition
  • Duplicate detection and handling

Automated Admission Letters

Generate professional provisional admission letters in PDF format for all admitted candidates with a single click. Bulk generation for entire programmes available.

  • Professional letterhead design
  • JAMB reference number included
  • Batch download as ZIP archive

Role-Based Access Control

Multi-level user permissions ensure that only authorized personnel can view, modify, or approve admission decisions. Complete audit logging for accountability.

  • Admin, Officer, Reviewer roles
  • Complete audit trail
  • Session security
